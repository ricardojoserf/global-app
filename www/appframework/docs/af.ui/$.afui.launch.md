#$.afui.launch();

```

This is callled when you want to launch afui.  If autoLaunch is set to true, it gets called on DOMContentLoaded.
If autoLaunch is set to false, you can manually invoke it.
 
```

##Example

```
 $.afui.autoLaunch=false;
 $.afui.launch();
 
```
