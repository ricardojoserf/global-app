#$.afui.updateBadge(target,value,[position],[color])

```

Update a badge on the selected target.  Position can be
  bl = bottom left
  tl = top left
  br = bottom right
  tr = top right (default)
 
```

##Example

```
 $.afui.updateBadge("#mydiv","3","bl","green");
 
```


##Parameters

```
target                        string
value                         string
position                      string=
color                         (string=|object)

```

##Returns

```
undefined
```
